Reflections

Reflection 1: Probabilistic AI

For this project, I considered how probabilistic AI could assist in web development. Tools like GitHub Copilot use predictive models to suggest code based on prior patterns, speeding up development while reducing errors. While Copilot helped me scaffold HTML, CSS, and Bootstrap components efficiently, I made deliberate choices to ensure accessibility, RTL support, and responsive design — areas where AI suggestions sometimes require human review. This project reinforced the importance of using AI as a partner rather than a replacement, especially when designing inclusive and globally accessible websites.

Reflection 2: Interview Question

Question: "How would you ensure a website is accessible to a global audience?"
Answer: I would implement semantic HTML, provide text alternatives for images, use color schemes with sufficient contrast, and design forms with proper labels and aria attributes. For global audiences, I would support multiple languages and include right-to-left (RTL) layouts for languages like Arabic or Hebrew. Testing tools like Lighthouse can help measure accessibility performance, and responsiveness across devices ensures usability for all users.

Reflection 3: LinkedIn Post

"Excited to share my latest web development project for Intel! I adapted a global landing page to support right-to-left languages, implemented accessible forms, and designed a responsive three-column layout using Bootstrap. This project strengthened my skills in inclusive design, responsive development, and using AI tools like GitHub Copilot to accelerate coding while maintaining quality. Accessibility and global usability are priorities in modern web design — proud to have applied them in this project!"
